Kita akan melanjutkan project kita sebelumnya, yaitu project membuat Program Konverter Suhu. Pada file TugasLuaOOP_NamaKamu.lua, kita telah membuat kelas suhu dan objek objSuhu. Sekarang coba kita praktikan konsep enkapsulasi dalam project kita kali ini. Ikuti intruksi berikut ini!

rn

Detail Tugas

rn
    rn
  1. Buatlah public variabel dengan nama ukuran_suhu bernilai 0 didalam kelas suhu.
    2. rn
  2. Buatlah public method setSuhu() di dalam kelas suhu.
    3. rn
  3. Tambahkan parameter _suhu_now ke dalam method setSuhu()
    4. rn
  4. Method setSuhu() berfungsi mengubah nilai variabel ukuran_suhu berdasar nilai parameter _suhu_now
    5. rn
  5. Buatlah public method getSuhu() didalam kelas suhu.
    6. rn
  6. Method getSuhu() akan mengembalikan nilai variabel ukuran_suhu .
    7. rn
  7. Buatlah public method showSuhu() di dalam kelas suhu
    8. rn
  8. method showSuhu() akan menampilkan output dari method getSuhu()
    9. rn
  9. Melalui objek objSuhu , panggil method setSuhu()  dan isi dengan parameter bertipe number (misal 30).
    10. rn
  10. Melalui objek objSuhu , panggil dan tampilkan nilai method showSuhu()  pada program.
    11. rn
